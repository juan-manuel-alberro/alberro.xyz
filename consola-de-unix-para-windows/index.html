Consola de Unix para Windows

10/08/2015 - #consola #desarrollo #linux #windows | 1 minuto de lectura

Uno de los grandes problemas que tiene Windows, desde la versión 3.1 hasta la recientemente lanzada versión 10, es que no cuenta con una buena consola. El famoso cmd es bastante limitado si se lo compara contra la consola / terminal de una plataforma Unix, ya sea Linux o Mac OS. Con este emulador vas a poder tener la misma consola de Unix sobre Windows y encima es portable.

Lo bueno de Cmder es que está basada sobre ConEmu, uno de los mejores emuladores para Windows y por supuesto, Cmder hereda todas las ventajas de él.

Además, Cmder cuenta con integración para Git poniendo adornos en la consola, como el nombre del branch, cambios hechos y demás mejoras. El diseño posee colores basados en el theme Monokai, el cual hace atractiva la consola para su uso diario.

 

Al estar basado en ConEmu, Cmder permite tener en una sola ventana muchas consolas como se puede ver en la captura de arriba. El diseño de cómo se pueden ubicar, está limitado a la imaginación del usuario que la usa 🙂
