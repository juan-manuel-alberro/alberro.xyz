Lanzador de aplicaciones al estilo Spotlight en Windows

10/05/2016 - #javascript #linux #open source #windows | 2 minutos de lectura

Apple tiene Spotlight y Alfred, Linux/Ubuntu tiene Unity y ahora Windows tiene Hain, el lanzador de aplicaciones y buscador de archivos open-source. Al mejor estilo Spotlight, Hain no es solo un buscador, es una herramienta que nos simplifica de manera notoria el uso de la computadora y nos da velocidad a la hora de ejecutar comandos.

Hain se ejecuta de manera muy sencilla, solo tenemos que descargarnos el archivo y abrir el .exe, por suerte no necesita instalación. Luego con solo presionar las teclas alt+space (se puede cambiar en caso de que no nos sea cómoda esa combinación de teclas) se nos abre una ventana para ingresar texto y ahí podemos buscar archivos o aplicaciones instaladas.

Posee tolerancia a los errores de escritura, es decir que si ingresamos la palabra WRD nos va a traer Word y todas las palabras y archivos que tengan un nombre similar. Funciona también con URLs, si escribimos una dirección web y presionamos enter nos va a abrir nuestro navegador por defecto con la URLs que ingresamos.

Podemos hacer uso de internet para usar Hain como diccionario con solo poner el ? delante de nuestra palabra a consultar como así también realizar operaciones matemáticas con solo escribir la fórmula.

Y lo más interesante, es que se puede extender el funcionamiento de Hain a través de plugins logrando llevar la configuración a lo que realmente necesités.

Hain está desarrollado en Javascript bajo la plataforma Electron, una herramienta que permite crear aplicaciones nativas para todos los OS disponibles con un mismo código fuente.
