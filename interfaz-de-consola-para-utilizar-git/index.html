Interfaz de consola para utilizar Git

18/07/2015 - #desarrollo #git #node #Software | 1 minuto de lectura

Git, el famoso software para control de código, puede ser utilizado desde la consola a través de diversos comandos. Ahora, con Git Commander, vas a poder utilizarlo en la consola pero a través de una interfaz gráfica bien definida.

Hasta el momento, Git Commander soporta las siguientes funcionalidades

  • git status
  • git add [files]
  • git reset — [files]
  • git commit [files]
  • git log
  • git reset
  • git diff file

Y en breve estarán incluyendo las funciones de git branch, git merge (handling conflict) y git push. A continuación vemos cómo funciona esta aplicación.

 

Los requisitos de software para que Git Commander funcione, son

  • git >= 2.4
  • nodejs >= 0.12
  • blessed >= 0.1.7
  • lodash >= 3.0

Y para su instalación solo tenemos que ejecutar los siguientes comandos en una consola, ya sea de Linux, Mac o Windows

npm install -g git-commander

Y luego para ejecutarlo

git-commander

Al estar instalado en forma global, git-commander queda disponible desde cualquier ubicación en la consola.
