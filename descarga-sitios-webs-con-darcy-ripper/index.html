Descarga sitios webs con Darcy Ripper

12/07/2015 - #Download #Software #URL | 1 minuto de lectura

Darcy Ripper es una aplicación gratuita para descargar sitios webs completos, pudiendo definir los niveles de profundidad que tiene que escavar. Se puede utilizar para bajar sitios y leerlos offline o simplemente investigar los recursos que posee. 

Los requerimientos que posee esta aplicación son prácticamente nulos, solo necesita tener instalado la versión de Java 1.6 en adelante. Es completamente independiente del sistema operativo que se corra, ya que está desarrollada en Java y funciona en Windows, OS X y Linux.

A su vez permite la configuración por cada uno de los proyectos que se manden a la cola de procesamiento, pudiendo definir a cada uno de ellos la profundidad de escaneo, el filtro de imágenes, la cantidad de descargas en paralelo y muchas opciones más! Ideal para entrar a esas webs que venden themes cof cof…
