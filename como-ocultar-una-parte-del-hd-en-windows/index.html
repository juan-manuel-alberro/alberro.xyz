Cómo ocultar una parte del HD en Windows

24/03/2017 - #windows #seguridad | 2 minutos de lectura

Hoy te voy a mostrar cómo esconder tu disco rígido sin usar ningún software de terceros, usando solamente las herramientas que nos da Windows. Es muy útil si tenés información importante que nadie querés que vea, ya sea por confidencialidad, fotos o simplemente porque tenés datos personales como extractos bancarios, contraseñas, etc y no querés a nadie revisando esto.

Hoy en día hay muchos programas en el mercado que se utilizan para ocultar la unidad pero en realidad no es necesario comprar, piratear o bajarlos, porque se puede hacer muy fácilmente mediante el uso de cmd. Esto ahorra dinero y tiempo.

Lo primero que voy a aclarar, es que no me responsabilizo si se produce alguna pérdida de datos o daño en el HD al hacer eso, si bien es imposible que suceda lo aclaro. Ahora bien, lo que tenemos que hacer es abrir una consola de Windows para poder ejecutar un par de comandos, para ello presionamos la combinación de teclas Win+R para que se abra el diálogo de Ejecutar Programas y ahí tipeamos cmd.

Paso 1

Ejecutar cmd

Paso 2

Escribir en la consola el comando diskpart y presionar enter, es probable que Windows nos solicite los permisos de admin para poder ejecutarlo

Paso 3

El comando previo abre una nueva ventana en donde tendremos a diskpart ejecutando, ahora escribimos list volume para conocer todos los discos y particiones que Windows detecta

Paso 4

Teniendo el listado de los discos y particiones, ahora nos resta elegir el que queremos ocultar. Para ello tipeamos select volume X donde X es la unidad a ocultar, en el ejemplo uso la unidad E.

Paso 5

Teniendo la unidad seleccionada, y sabiendo que no podemos fallar, ahora sí procedemos a ocultar la unidad ejecutando remove letter X donde X es la unidad a ocultar, en el ejemplo sigo usando la unidad E. Si bien el comando es remove, no se preocupen que los datos no se borran ni se alteran.

Y la unidad desaparece del Explorador de Windows

Eso es todo, ahora si queremos restaurar la unidad para que sea visible, lo que tenemos que hacer es ejecutar el siguiente comando para volver todo a default

assign letter X donde X es la unidad a mostrar

Eso es todo, bien sencillo y sin instalar niniguna aplicación externa. A partir de ahora ya sabés cómo ocultar tu información importante y lo que querés ocultar…
