Guarda tus links de una forma original gracias a Stash

Si sos realmente un nerd, ya sabés de qué se trata esta aplicación ya que hace analogía al comando stash de git y que seguramente su autor puso ese nombre en honor a ese comando. Si no sos un nerd, entonces paso a explicarte que esta aplicación sirve para guardar las páginas webs y poder consultarlas en otro momento, algo parecido a lo que sería Pocket por ejemplo. Stash sabe si lo que estamos guardando es una noticia, una receta de cocina o un video de Youtube, por ejemplo; consigue identificar el contenido y lo guarda con una estructura predefinida para que no se pierda entre tantos links que puedas guardar.
