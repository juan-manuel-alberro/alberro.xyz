Consola de Unix para Windows

Uno de los grandes problemas que tiene Windows, desde la versión 3.1 hasta la recientemente lanzada versión 10, es que no cuenta con una buena consola. El famoso cmd es bastante limitado si se lo compara contra la consola / terminal de una plataforma Unix, ya sea Linux o Mac OS. Con este emulador vas a poder tener la misma consola de Unix sobre Windows y encima es portable. Lo bueno de Cmder es que está basada sobre ConEmu, uno de los mejores emuladores para Windows y por supuesto, Cmder hereda todas las ventajas de él.
