Sitecake – CMS para crear sitios estáticos

Sitecake es un gestor de contenidos, Content Management System en inglés, que permite crear y administrar páginas webs estáticas, sin necesidad de utilizar una base de datos para almacenar los posts. Soporta todos los frameworks más conocidos del mercado como Bootstrap, Foundation y HTML5 Boilerplate, entre otros.
