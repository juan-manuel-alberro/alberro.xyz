Librerías de íconos para aplicaciones

02/07/2015 - #gratis #iconos #recursos | 2 minutos de lectura

Ya sea porque estás desarrollando una aplicación web, una presentación para la facultad, un juego o lo que sea, vas a necesitar íconos para hacer tu trabajo más vistoso. A continuación les traigo una lista de recursos gratuitos para embeber en sus proyectos.

Font Awesome

Si hablamos de librerías de íconos, Font Awesome es probablemente una de las más conocidas. Una fuente con +510 íconos y una muy buena documentación van a hacer que tu trabajo llame realmente la atención.

Open Iconic

Un set con +220 íconos en formato SVG, webfont y raster.  Muy buena documentación para todos sus formatos, ideal para aplicaciones web.

Typicons

Una fuente con +330 íconos a nivel de pixel perfect, según sus creadores. La documentación no es tan grande pero alcanza como para aprender a usar esta librería. Los íconos tienen un toque particular, un no sé qué que van a caracterizar tu proyecto.

Entypo

Una suite con +400 íconos, muy bien diseñados y en diferentes tamaños. La documentación no existe, solo una FAQ que tiene un poco de información. Esta librería ya no viene más como una fuente, vale la aclaración.

Linecons

Del juego de palabras line+icons, íconos y líneas. Cuenta con un pequeño set de solo 40 íconos pero todos muy lindos y bien pensados. Se pueden bajar uno a uno o todos juntos, además cuenta con la posibilidad de embeberlos como BASE64 entre otros formatos.

IcoMoon

Además de ser un set de íconos, cuenta con una app que permite seleccionar los íconos que querramos y exportarlos en una fuente, ideal para no sobrecargar el peso del archivo si solo necesitamos un par de íconos.
