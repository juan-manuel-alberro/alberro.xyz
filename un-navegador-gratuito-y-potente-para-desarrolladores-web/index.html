Un navegador gratuito y potente para desarrolladores web

23/05/2016 - #desarrollo #navegador #web | 2 minutos de lectura

Blisk es un navegador gratuito creado exclusivamente para personas que se dedican al desarrollo web. Este navegador tiene la particularidad que puede ser usado desde cualquier etapa de desarrollo, desde la creación de prototipos hasta el testing en diferentes devices a través de los emuladores integrados que posee.

Una de las características nativas que posee, y que en otros navegadores solo se logra añadiendo plugins, es la sincronizar el scroll en diferentes vistas de manera que si nos movemos dentro de un sitio web, la vista en el emulador también se mueve y de esa manera podemos comparar como responde el diseño desde una misma ubicación.

Blisk trae incorporado emuladores para diversos tipos de dispositivos como ser celulares, tablets y hasta PC de escritorio. Por nombrar algunos, trae los emuladores pre-instalados de:

  • iPhone 4, 5, 6 y 6 Plus
  • Nokia N9 y Lumia 625
  • Samsung S3, S4 y S6
  • iPad Mini
  • iPad Air Retina
  • Dell Inspiration 5748
  • entre otros

    • Además trae lo que se conoce como watcher, a medida que uno realiza un cambio en el código de nuestra aplicación o página web, automáticamente Blisk refresca la pantalla para mostrarnos la última versión. Funciona para las extensiones html, css, js, saas, cshtml, php y muchas más.

    Incorpora dentro de su motor, un Analytics para detectar errores de Javascript y para realizar recomendaciones sobre cómo mejorar la velocidad de la web, sobre si tu web es compatible con otros navegadores como Chrome, Firefox o IE/Edge, y además verifica enlaces rotos para recursos css y fuentes.

    Te dije que captura pantallas en todos los devices y vistas abiertas con un solo clic? Ideal para los testers / QA.

    Y por si querías más, en un futuro no muy lejano, van a implementar las integraciones con otros sitios web para poder subir o reportar errores directamente desde el browser a Jira, Github y otras herramientas.

    Lo negativo, y que por suerte algo temporal, es que sólo está disponible para el sistema operativo Windows (WTF right?). Para Linux y Mac todavía no hay versión disponible para descargar, pero están trabajando en eso ya que si no lo hacen pierden muchísimo mercado ya que son los dos sistemas operativos más utilizados para el diseño y desarrollo web.
