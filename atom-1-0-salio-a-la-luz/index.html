¡Atom 1.0 salió a la luz!

25/06/2015 - #editor #IDE #open source | 1 minuto de lectura

Desde hoy ya se puede descargar la versión 1.0 del entorno de desarrollo Atom, el IDE gratuito y hackeable de Github. Para los que no lo conocen, este editor es la competencia más fuerte que posee SublimeText, con la diferencia de que Atom es completamente gratuito.

Siendo la competencia de ST, Atom no se podía quedar atrás e incorpora dentro de su motor la capacidad de agregar temas (themes), paquetes de lenguajes y funciones (packages) y diferentes customizaciones para poder configurar el editor como a uno más le quede conveniente.

Para la release 1.0, los muchachos de Atom lanzaron un video un poco particular, basado en la década del ’60, aproximadamente, donde se muestra a Atom corriendo en una terminal muy antigua.

Este IDE funciona en todas las plataformas más conocidas ya que es cross-platform: Windows, OSX y Linux. Posee capacidades de autocompletar a medida que el desarrollador tipea comandos, navegador de archivos, capacidad para dividir el IDE en múltiples columnas y muchas opciones más.
